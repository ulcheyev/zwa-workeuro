Detailed Description

Kontrola nastaveni tikenu. Na webu jsou dva typy formulare. Jeden posila data Get, ostatni Post. Pokud se generovany token neshoduje s SESSION tokenem => presmerovani na stranku s chybou.

if(isset($_POST['csrf_token'])){
if($_POST['csrf_token'] != $_SESSION['csrf_token']){
header('Location: ../errors/csrf-token-error.php');
exit();
}
}else{
if(isset($_GET['csrf_token'])){
if($_GET['csrf_token'] != $_SESSION['csrf_token']){
header('Location: ../index.php');
exit();
}
}
